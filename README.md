## Running Locally

- Clone this Repository

```bash
git clone = https://github.com/Eshwarsai-07/krishiPrabandhan.git
```

- Create `.env` file in root directory

```bash
MONGO_URL=
JWT_SECRET=
```

- Create `.env` file in client directory

```bash
VITE_FIREBASE_API_KEY= 
```

- In  `/` Run 
```bash
 npm run dev
```


- In `/client` Run
```bash
 npm run dev
```
